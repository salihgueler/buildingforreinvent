Create a new React + TypeScript project using Vite with the following setup:

- Tailwind CSS v4 for styling
- React Router v7 for navigation
- shadcn/ui components (Button, Input, DropdownMenu)
- next-themes for dark/light mode
- sonner for toast notifications
- lucide-react for icons
- react-hook-form with zod for form validation

Initialize the project with a dark theme gradient background (slate-900 to slate-800).
